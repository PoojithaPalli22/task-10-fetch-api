# Task 10: Country Information using Fetch API

## ðŸ”¹ Project Overview
This project demonstrates **client-side consumption of a public API** using **JavaScript Fetch API**.  
The app allows users to enter a country name and fetches details like:

- Country Name
- Capital
- Population
- Region
- Flag

The data is fetched from the **REST Countries API** (https://restcountries.com).

---

## ðŸ”¹ Features
- Dynamic country search by name
- Displays country details on the webpage
- Shows loading indicator while fetching data
- Handles errors (e.g., "Country not found")
- Case-insensitive search
- Responsive UI

---

## ðŸ”¹ Tools Used
- **Languages:** HTML, CSS, JavaScript
- **API:** REST Countries API (public, no authentication)
- **Code Editor:** VS Code
- **Browser:** Chrome / Edge / Firefox

---

## ðŸ”¹ How to Run
1. Clone or download this repository.
2. Open `index.html` in a browser.
